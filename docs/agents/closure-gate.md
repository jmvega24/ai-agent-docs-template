# Closure Gate

Formal closure gate for project work.

## Status meanings

- `implementation in progress`: work is not yet auditable.
- `implementation ready`: implementation exists, but review or sync is still pending.
- `pending audit`: implementation exists, but it has not been reviewed.
- `pending sync`: implementation was reviewed, but documentation or planning artifacts are not aligned.
- `closed`: implementation, evidence, and documentation are aligned.

## Minimum gate for `closed`

Before using `closed`, confirm:

- observable result exists;
- validation evidence exists;
- active plan is updated;
- relevant logs are updated;
- context is aligned;
- affected docs are aligned;
- no unresolved contradiction blocks closure.
