# Template Index

This folder is a neutral documentation starter for any software project.

## Included areas

- agent governance
- operational context
- architecture and conventions
- contracts
- planning
- logs
- API references
- ADRs
- UX guidance

## Replace first

- `[PROJECT_NAME]`
- `[TEAM_NAME]`
- `[RUNTIME]`
- `[FRAMEWORK]`
- `[DATABASE]`
- `[AUTH_STRATEGY]`

## Remove if not needed

- `docs/contracts/backend-external/`
- `docs/contracts/internal/`
- `docs/ux/`
- `docs/api/postman-collection.json`
