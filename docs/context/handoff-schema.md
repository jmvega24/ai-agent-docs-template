# Handoff Schema

Use this schema when the orchestration layer delegates work to the execution layer.

```text
IMPLEMENTATION TASK
Goal:
Work item:
Business or product context:
Relevant files:
Files to avoid touching:
Constraints:
Acceptance criteria:
Tests required:
Docs to update:
Output expected:
  - List of changed files
  - Execution summary
  - Test commands run + pass/fail summary
  - Open risks / assumptions
```

## Notes

- Keep handoffs short.
- Include only the context needed to execute the task.
- State exclusions explicitly.
